# Organizador de Arquivos em Pastas

Este projeto é um **organizador automático de arquivos**, desenvolvido em Python, criado para treinar lógica, automação e boas práticas de programação.

O script monitora uma pasta e distribui automaticamente cada arquivo para seu tipo correto (imagens, vídeos, documentos, PDFs, ZIPs, etc.), mantendo tudo organizado de forma simples e eficiente.

---

## Funcionalidades
- Identifica arquivos pelo tipo/extensão  
- Cria automaticamente as pastas necessárias  
- Move cada arquivo para sua categoria correta  
- Evita duplicações  
- Organização rápida e automatizada  

---

## Tecnologias utilizadas
- **Python 3**
- Manipulação de arquivos com `os` e `shutil`
- Estrutura de dicionário para mapeamento de extensões

---

2. Coloque o script `organizer.py` dentro da pasta que deseja organizar.
